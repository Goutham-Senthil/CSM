# Conditioning Sign on Magnitude

This repository is recreation of the paper "A new decomposition approach to modeling financial returns: Conditioning sign on magnitude".

It can be found [here](https://arxiv.org/abs/2606.04153).

## Dataset

The dataset used in this paper was curated by Amit Goyal.

The dataset can be found [here](https://sites.google.com/view/agoyal145).

It is an excel file found under the subheading "A Comprehensive 2022 Look at the Empirical Performance of Equity Premium Prediction".

Since I am only interested in the monthly data, I have used the file "Data2024.xlsx - Monthly.csv".
